ML Theory

Linear Regression

Linear Regression fits a line to predict Y from X. The line is defined as Y = slope * X + intercept.

Classification

Classification separates data into categories. Here we use a simple line boundary between two classes.

Gradient Descent

Gradient Descent iteratively updates model parameters to reduce error between predicted and actual values.
